Welcome to the API

This API provides endpoints for managing users and their products. Below you'll find the available endpoints and their descriptions.

  • GET /users
    Retrieve a list of all users. Supports pagination, filtering, and sorting.
    Query parameters:
    • page: Page number (default: 1)
    • limit: Items per page (default: 10)
    • sortBy: Sort field (default: id)
    • sortOrder: Sort direction (asc/desc)
  • GET /users/:id
    Retrieve a specific user by their ID.
  • POST /users
    Create a new user. Required fields in request body:
    • name: User's name
    • email: User's email
  • GET /users/:id/products
    Retrieve all products belonging to a specific user.
  • POST /users/:id/products
    Add a new product for a specific user. Required fields in request body:
    • name: Product name
    • price: Product price
    • description: Product description
